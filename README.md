# Clasificador de Objetos con IA

Proyecto de **Inteligencia Artificial** para clasificar imágenes de objetos de escritorio usando **Deep Learning** y **MobileNetV2**.

## Clases
- Calculadora
- Cartuchera
- Engrapadora

## Contenido
- **app.py**: Aplicación en Streamlit para realizar predicciones desde imágenes.
- **modelo_objetos_tf.h5**: Modelo entrenado listo para inferencia.
- **Notebook de Colab**: Entrenamiento del modelo, preparación de datos y pruebas.

## Tecnologías
TensorFlow · Keras · Streamlit · Python · Google Colab
